# Otterize Network Mapper Visualizer

<img width="3558" height="1960" alt="image" src="https://github.com/user-attachments/assets/a4b94848-e7bb-4ff5-b136-b6dd06a4cbd9" />

This is the companion repo to [this video](https://example.com/todo). You should consume this repo alongside the video.

The code here takes a YAML output from the Otterize network mapper and visualises it in Python.

PRs and improvements welcome!

## Installation

```
python -m venv .
. bin/activate
pip install -r requirements.txt
# Now generate the data.yaml network output using
# the otterize CLI (see video)

# Visualise the data.yaml file
python app.py
```
